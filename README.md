# MERN Boilerplate
Reuseable MERN template   

### Installing

```
 
clone https://github.com/BFronz/mern-boilerplate.git

npm install @ client & server 

Run  npm run dev

```

## Built With
* Mongodb
* Express
* React
* Node.js
* Redux
* Materialize-CSS
* Json Web Token
